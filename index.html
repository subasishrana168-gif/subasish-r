E-Commerce Shopping Cart

This is a simple command-line shopping cart application written in C.

Features

  • View product catalog
  • Add items to cart
  • Remove items from cart
  • View cart contents
  • Checkout with tax calculation

How to Use

This is a C program. To run it:

  1. Download shopping_cart.c
  2. Compile with GCC: gcc shopping_cart.c -o shopping_cart.exe
  3. Run: ./shopping_cart.exe
